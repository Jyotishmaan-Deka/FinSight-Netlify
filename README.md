# FinSight

A modern personal finance companion app for Android — built to turn everyday money management into a simple, insightful habit.

## About

FinSight helps users track spending, manage budgets, and work toward savings goals through a clean, dark-first interface. The app focuses on giving people clear, actionable insight into their financial habits rather than just raw numbers — combining interactive dashboards, goal tracking, and spending breakdowns in one place.

## Features

- **Dashboard & Insights** — Visual breakdown of income, expenses, and savings rate, with daily spending trends and category-level analysis
- **Transactions** — Live search, filterable by All / Income / Expense, with color-coded entries for quick scanning
- **Goals System** — Four goal types with progress tracking, deadlines, and custom color tagging
- **Currency Converter** — Live exchange rates across 30+ currencies
- **Secure Auth** — Firebase-backed authentication
- **Material Design 3** — Clean, modern UI built on Android's latest design system

## Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose, Material Design 3
- **Backend:** Firebase Authentication, Cloud Firestore
- **Data:** Live exchange rate API
- **Notifications:** Local notifications for reminders and alerts
